# Cravio — Gamified Food Delivery Interface

An interactive food delivery platform featuring gamified engagement mechanics and role-based dashboards for different user types.

## 🚀 Overview

Cravio focuses on enhancing user engagement in food ordering systems through reward-based interaction and game mechanics.

## ✨ Features

- Role-based dashboards (customer, delivery partner, admin)
- Gamified reward system
- Browser mini-game with redeemable points
- Interactive UI workflows
- Responsive design

## 🏗 Tech Stack

- React
- Node.js
- HTML
- CSS

## 🧠 Engineering Highlights

- Gamification integration into e-commerce workflows
- State-driven reward logic
- Reusable UI components
